\subsection{Detailed Description}
Macros das funções de depuração. 

Macros que serão usadas nos programas desenvolvidos ao longo dos exemplos. Estas macros podem receber um número variável de parâmetros através de uma string de formatação como no \char`\"{}printf\char`\"{}. Seguem-\/se alguns exemplos\+: 
\begin{DoxyCode}
\hyperlink{debug_8h_a96dd473db0b3d10bd43390cdacb00120}{DEBUG}(\textcolor{stringliteral}{"i = %d e f=.2f%"}, i, f);
\hyperlink{debug_8h_a8f448861da74111034b91e293ab3a278}{ERROR}(\textcolor{stringliteral}{"%s"}, msg);
\end{DoxyCode}
